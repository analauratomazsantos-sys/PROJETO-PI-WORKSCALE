# WorkScale — Gestão de Escalas

> Projeto Integrador DSM/FATEC Franca — plataforma web para gerenciamento de escalas de trabalho.

## Descrição

O WorkScale é uma aplicação front-end simples que ajuda empresas a organizar escalas, turnos
e colaboradores em um ambiente digital leve. Este repositório contém a versão estática do
site (HTML/CSS/JS) usada como protótipo do projeto.

## Funcionalidades principais

- Cadastro da empresa (página de formulário)
- Cadastro de funcionários
- Montagem e visualização de escalas
- Interface responsiva para navegação básica

## Estrutura do projeto

- `index.html` — página inicial
- `empresa.html` — cadastro da empresa
- `escala.html` — criar/montar escala
- `login.html` / `cadastrolog.html` / `AlterarSenha.html` — autenticação básica
- `css/` — arquivos de estilos (por exemplo `style.css`)
- `js/` — scripts JavaScript (por exemplo `script.js`)
- `imagens/` — imagens usadas no site

## Como executar

1. Abra o arquivo `index.html` diretamente no seu navegador (Chrome, Edge, Firefox).
2. Não é necessário servidor; basta abrir o arquivo localmente para navegar pelas páginas.

Opcional: para testar caminhos relativos em um servidor local, execute um servidor HTTP simples, por exemplo:

```bash
# Python 3
python -m http.server 8000
```

E depois acesse `http://localhost:8000` no navegador.
